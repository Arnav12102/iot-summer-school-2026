# LED Blink Project

## Project Description

This project demonstrates the basic LED Blink program using Arduino. The built-in LED connected to digital pin 13 turns ON and OFF repeatedly with a fixed time interval. This project is commonly used as the first Arduino program to verify that the board and development environment are working correctly.

---

## Hardware Required

- Arduino Uno
- USB Cable
- Computer with Arduino IDE
-  External LED
-  220Ω Resistor
-  Breadboard
- Jumper Wires

---

## Circuit Diagram Description
use tinked cad for the simulation purposes

### Using External LED

Connect the components as follows:

Pin 13 → 220Ω Resistor → LED Anode (+)

LED Cathode (-) → GND

---


## Output in tinker cad

- The LED connected to digital pin 13 turns ON for one second.
- The LED turns OFF for one second.
- This blinking process repeats continuously.
